This is a draft of an extension for OpenID Connect to supports quotas for the number of active sessions (at an RP side).

The proposal was made in [issue2184](https://bitbucket.org/openid/connect/issues/2184/openid-connect-and-user-session-quotas-at) and [one](https://lists.openid.net/pipermail/openid-specs-ab/2025-September/010958.html) of the AB/Connect WG calls.
